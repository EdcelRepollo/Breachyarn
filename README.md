# Breachyarn
Cyber security tasks for prelim
1-Month Cybersecurity CTF Roadmap
Overview

This roadmap focuses on learning and practicing web security vulnerabilities through CTF challenges. The first weeks focus on understanding basic concepts, while the later weeks focus on hands-on practice and solving challenges such as TryHeartMe, Love Letter Locker, and Signed Messages.
**Week 1 — Web Security Fundamentals**
Goal: Understand basic web application security concepts.

**Topics**
Introduction to Web Security
Common Web Vulnerabilities
HTTP and HTTPS basics
Cookies and Sessions
Authentication and Authorization

**Tools to Learn**

Browser Developer Tools
CyberChef
JWT Decoder
Burp Suite (basic)

**Practice**
Inspect webpage source code
Analyze cookies and session tokens
Decode encoded data
Practice beginner web CTF challenges

**Week 2 — Authentication & Access Control**

Goal: Understand how weak authentication can be exploited.

**Topics**
JSON Web Tokens (JWT)
Session Management
Access Control
Role-based permissions
Vulnerabilities to Study
JWT Manipulation
Insecure Authentication
Privilege Escalation

**Practice**

Modify tokens and analyze their structure
Use browser developer tools to inspect cookies
Solve CTF challenges similar to TryHeartMe

**Week 3 — Web Application Vulnerabilities**

Goal: Learn how attackers access unauthorized data.

**Topics**

Insecure Direct Object Reference (IDOR)
Parameter manipulation
URL tampering
Access control flaws

**Practice**

Identify sequential IDs in URLs
Attempt ID modification to test authorization
Solve challenges similar to Love Letter Locker
**Expected Outcome**

Ability to detect weak access control
Understand how attackers retrieve restricted data

**Week 4 — Cryptography & Message Integrity**

Goal: Understand how message authentication and signatures work.

Topics

Digital signatures
Message verification
Hashing and encryption basics

**Practice**

Analyze signed messages
Understand how signatures validate data
Solve cryptography challenges similar to Signed Messages
Mini Project
Document one solved CTF challenge

Include:
Steps taken
Tools used

Vulnerability discovered
Solution

**Expected Outcomes**

After completing this roadmap, you should be able to:
Understand common web application vulnerabilities
Analyze cookies, tokens, and authentication mechanisms
Identify IDOR and access control flaws
Understand basic cryptography and message validation
Solve beginner to intermediate CTF challenges
Write structured cybersecurity reports
